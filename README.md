# Gym Registro

App web/PWA gratuita para registrar entrenamientos desde el teléfono.

## Qué incluye

- Carga de fecha, ejercicio, series, repeticiones, peso y notas.
- Desplegable de ejercicios principales.
- Opción `+ Nuevo ejercicio`, guardada para próximas cargas.
- Historial por fecha con editar, borrar y duplicar.
- Progreso por ejercicio con último peso, mejor peso, registros, volumen y gráfico.
- Exportar CSV.
- Exportar/importar backup JSON.
- Funciona sin servidor y guarda datos en el teléfono con `localStorage`.

## Subir a GitHub Pages

Subí a la raíz del repositorio estos archivos y carpetas:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `sw.js`
- carpeta `assets`

Después activá GitHub Pages desde Settings > Pages > Deploy from a branch > main > / root.
